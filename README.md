# Battleship Game

## Overview
This project is an implementation of the classic Battleship game built with JavaScript. The game allows a player to battle against a computer by attacking coordinates on a gameboard until all ships of one side are sunk.

## Features
- Classic Battleship gameplay
- Player vs computer
- Turn-based attacks
- Random computer moves
- Game ends when all ships of a player are sunk

## Technologies Used
- JavaScript (ES6+)
- Jest for unit testing
- HTML
- CSS

## Concepts Practiced
- Test Driven Development (TDD)
- Writing unit tests with Jest
- Modular JavaScript
- Classes / factory functions
- Game logic and state management

## Purpose
The purpose of this project was to practice structuring JavaScript applications and writing tests to validate application logic.
